# TalkFellas - Professional Conversation Support BPO

A modern, responsive single-page website for TalkFellas, a professional conversation-support BPO focused on increasing customer talk-time through structured empathy, active listening, and high-quality human engagement.

## Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Fast Loading**: Optimized CSS and minimal JavaScript
- **SEO Friendly**: Proper meta tags and semantic HTML structure
- **Accessible**: Built with accessibility best practices

## Services Highlighted

- Talk-Time Optimization
- Emotionally Intelligent Support
- Process-Driven Operations
- Customer Conversation Support
- Talk-Time Enhancement Programs
- Agent Training & QA

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)
